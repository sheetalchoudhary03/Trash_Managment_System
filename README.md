# ♻️ Trash Management System with Analytics Dashboard

## 📌 Introduction
The Trash Management System is a Python-based application designed to efficiently manage and analyze waste collection data. It provides a structured way to handle different categories of trash, track industry deals, and generate insights through an interactive analytics dashboard.

The system combines data management with visualization tools to improve decision-making and promote efficient waste handling practices.

---

## 🎯 Objective
- To manage waste collection data efficiently  
- To track distribution of waste across industries and storage  
- To analyze revenue and performance using visual dashboards  
- To provide a user-friendly system for waste monitoring  

---

## ✨ Features
- 📊 Analytics Dashboard with multiple graphs  
- ♻️ Category-wise waste management  
- 🏭 Industry client handling and deal tracking  
- 📈 Revenue trend visualization  
- 📉 Distribution and performance analysis  
- 💾 Data export in JSON format  
- 🖥️ GUI using Tkinter  
- 📊 Graphs using Matplotlib & Turtle  

---

## 🛠️ Tech Stack
- **Language:** Python  
- **GUI:** Tkinter  
- **Visualization:** Matplotlib, Turtle  
- **Data Handling:** JSON  

---

## ⚙️ How It Works
1. User enters trash data by category  
2. System distributes waste to industry clients or storage  
3. Deals and revenue are recorded  
4. Analytics dashboard generates:
   - Category comparison  
   - Revenue trends  
   - Distribution analysis  
   - Performance charts
